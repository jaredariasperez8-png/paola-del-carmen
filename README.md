# paola-del-carmen
1. Uso de cookies

Las cookies son pequeños archivos que un sitio web guarda en el navegador del usuario. Se utilizan para almacenar información como preferencias, idioma o datos de inicio de sesión, haciendo que la navegación sea más rápida y personalizada.

2. Creación de sesiones

Las sesiones permiten que un sitio web identifique al usuario mientras navega. Gracias a ellas, el usuario puede permanecer con la sesión iniciada sin tener que ingresar sus datos en cada página que visita.

3. Eliminación de sesiones

La eliminación de sesiones ocurre cuando el usuario cierra sesión (logout) o cuando la sesión expira. Esto elimina la información temporal del usuario y evita que otras personas puedan acceder a su cuenta desde el mismo dispositivo.
